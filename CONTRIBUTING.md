# Contributing

Raincloud is a personal hobby project and not the next cloudflare, but i will happily accept PRs and fix issues. \
If you are going to make a PR, please lint it with prettier and test your code first

## AI Code

Do not submit PRs that are primarily AI-generated. AI-assisted code is fine, but i will not merge vibecoded ai slop.
